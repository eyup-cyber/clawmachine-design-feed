# Silkscreen (retro-silkscreen-font)

**Upstream:** https://github.com/googlefonts/silkscreen (origin: Jason Kottke, 2001)
**License:** SIL Open Font License 1.1 — see OFL.txt

## What it looks like
Classic web-pixel font. 4/5px-high bitmap-style at native rendering. The canonical
"pixel font for small-size interfaces" choice from the early 2000s web. Readable down
to 8px. Shipped in Roman, Bold, Expanded, Expanded Bold.

## What it offers
- ./fonts — TTF distribution
- ./sources — .sfd FontForge sources
- ./scripts — build pipeline
- AUTHORS.txt, CONTRIBUTORS.txt, README.md, Makefile
- OFL-licensed, free to embed and modify

## Clawmachine fit
HIGH. OFL-licensed pixel font aligns perfectly with the OFL-only font doctrine (per
project_font_decisions_2026_04_20). Use for very small UI labels, monospace-adjacent
surfaces, or accent text where Chicago-Kare feels too chrome-specific. Complements
Satoshi + IBM Plex + JetBrains in the approved OFL set by filling the "pixel" slot.
