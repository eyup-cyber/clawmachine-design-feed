# Dead Space — diegetic UI (RIG spine, hologram menus)

**Clawmachine-fit rating:** S

**Notes:** The gold standard for 'HUD is the character's world'. Not aesthetically retro, but philosophically aligned: no chrome sitting on top, UI lives in-world. Blue/white holograms, thin weights.

**Credits / sources:** EA/Visceral, Polygon (Dino Ignacio GDC talk), iABDI, Gamer Rant, r/Design.
