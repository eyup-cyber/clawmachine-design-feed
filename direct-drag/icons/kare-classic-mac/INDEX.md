# Classic Mac Icons (retro-kare-classic-mac-icons)

**Upstream:** https://github.com/thomasareed/Classic-Mac-icons
**License:** NOT DECLARED — no LICENSE file present. Treat as all-rights-reserved until confirmed.

## What it looks like
Pixel-perfect recreations of original Classic Mac icons in both SVG (scalable) and PNG.
Direct Kare-lineage (the original Mac icons she designed). Intended by the author as
sticker source material but structured as a reusable icon set.

## What it offers
- ./svg — vector recreations
- ./png — raster at sticker-ready sizes
- stickers.jpeg preview

## Clawmachine fit
PRIMARY for the System-1-through-7 1-bit icon aesthetic — more complete than system.css's
bundled 3 dialog icons. BLOCKED by license status: reference only until author clarifies.
Consider opening an issue or emailing the author if anything here ends up in a shipped
build. For internal consolidate/inventory work this is fair use; any export needs sign-off.
