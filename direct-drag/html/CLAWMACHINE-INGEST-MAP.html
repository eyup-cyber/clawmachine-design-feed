clawmachine
→ claude design

A macOS SwiftUI app for second-hand tech-electronics arbitrage. Not a SaaS. Not a team tool. Not consumer. One UK operator, ten-hour days at a single dense terminal. Every pixel earns its keep — no placeholders, no narration, no gamification.

This 1-pager points you at the aesthetic ground-truth and the 7-page example. The operator's codex labs (chromatic-lab: 62 complete colour systems; style-lab: 20 UI style systems) are the canonical reference for density, type ramp, and card grammar. Derive from them — do not compete. The authored pages file renders all seven app pages in that vocabulary for direct side-by-side comparison. Together they form the whole brief.

Stack: macOS SwiftUI · Swift 6.3 · Foundation + SwiftUI only (no AppKit, no SF Symbols). Dark-dense default but the palette engine drives 62 schemes; per-page accents; 16x16 pixel icon grammar; beveled chrome; tabular numerics. Composite idiom: Mac OS 9 + Bloomberg Terminal + Fallout Pip-Boy rendered in operator-authored palettes. Apple liquid-glass is explicitly the antithesis.

Hard rules · non-negotiable from operator doctrine
DO
the frame
Dark, dense, data-first foundation. Beveled borders (highlight TL + shadow BR, solid colours). Tabular numerics everywhere a number appears. Per-page accents from active scheme's pg.* object. Spacing from {4, 8, 12, 16, 24, 32} only. Radius ≤ 4px. Grain overlay at 1.5%. Motion: standard ease-out for constrained, easeOutBack only on open surfaces.
DO NOT
the antithesis
No liquid-glass / translucency / blur / backdrop-filter. No gradients beyond a bevel. No SF Symbols. No Title Case. No gamification (win rate, streaks, XP, leaderboards). No developer-voice UI copy. No AI-generated illustration filler. No default-visible dismissed rows. No Matter font. No sell-to-CeX or sell-to-BS-Exchange. No dead UK vendors (Decluttr, Mazuma).
