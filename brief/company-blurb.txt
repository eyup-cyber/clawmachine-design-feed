clawmachine — a dense, data-first macOS SwiftUI desktop app for UK second-hand tech-electronics arbitrage. One solo operator, no team, ten-hour days at one trading terminal. The app runs on the operator's MacBook and talks over Tailscale to two headless nodes: a Raspberry Pi (the SCOUT — continuously scrapes eBay / CeX / Gumtree / Facebook Marketplace / Shpock / Preloved / Wholesale Clearance / musicMagpie / BS Exchange behind Decodo UK proxies) and a Mac Studio (the BRAIN — runs the local LLM and the deal-evaluator that scores each candidate against a multi-source floor-price aggregate and the eBay sold-comps median). The APP is the operator's cockpit: approve / skip deals, track inventory, watch ROI, configure channels, chat with the brain. The operator buys, fixes, resells on eBay.

Runtime is OpenClaw (agentic framework wiring brain ↔ scout ↔ app + tool registry + chat loop + BlueBubbles outbound). Claude Design does not need to generate runtime code; the vocabulary (brain / scout / app / agentconsole / mesh) shows up as VISIBLE UI LABELS and must land with meaning in the design.
